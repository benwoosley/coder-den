  1. Mean of boolean list

    In python I'm used to doing something like np.mean([True, False, True......]) to compute the fraction of True values in a list. Is there some equivalent in mathematica? Using Cases or Count seems way too verbose


    tags: python

Answers

  1. Use Boole to convert truth values to numbers.
    Mean@Boole@{True, False, True} (* 2/3 *)

    John Doe

  2. list = {True, True, False, True, False, True, False, True, True, True}
    Using mean gives:
    Mean[list] Out: 1/10 (3 False + 7 True)
    Then you can replace False and True with 0 and 1 respectively:
    Mean[list] /. False -> 0 /. True -> 1 Out: 7/10
